  • SDAdjustSVGContentMode

    Adjust SVGKImage‘s viewPort && viewBox to match the specify contentMode of view size.

    Note

    Though this util method can be used outside this framework. For simple SVG image loading, it’s recommaned to use sd_adjustContentMode property on SVGKImageView+WebCache.

    Declaration

    Objective-C

    extern void SDAdjustSVGContentMode(SVGKImage *_Nonnull image,
                                   UIViewContentMode contentMode,
                                   CGSize viewSize)

    Swift

    func SDAdjustSVGContentMode(_ image: SVGKImage, _ contentMode: UIView.ContentMode, _ viewSize: CGSize)

    Parameters

    image

    SVGKImage instance, should not be nil.
    contentMode

    The contentMode to be applied. All possible contentMode are supported.
    viewSize

    Target view size, typically specify the view.bounds.size.
