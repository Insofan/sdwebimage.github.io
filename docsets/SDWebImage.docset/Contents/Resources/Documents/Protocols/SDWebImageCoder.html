SDWebImageCoder

@protocol SDWebImageCoder <NSObject>

This is the image coder protocol to provide custom image decoding/encoding. These methods are all required to implement.

Note

Pay attention that these methods are not called from main queue.

    Swift

    func decompressedImage(with image: NSImage?, data: AutoreleasingUnsafeMutablePointer<NSData?>, options optionsDict: [String : NSObject]? = nil) -> NSImage?

    Parameters

    image

    The original image to be decompressed
    data

    The pointer to original image data. The pointer itself is nonnull but image data can be null. This data will set to cache if needed. If you do not need to modify data at the sametime, ignore this param.
    optionsDict

    A dictionary containing any decompressing options. Pass {SDWebImageCoderScaleDownLargeImagesKey: @(YES)} to scale down large images

    Return Value

    The decompressed image
