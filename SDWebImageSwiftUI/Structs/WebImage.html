  • animated(_:)

    Make the image to support animated images. The animation will start when view appears, and pause when disappears.

    Note

    Currently we do not have advanced control like binding, reset frame index, playback rate, etc. For those use case, it’s recommend to use AnimatedImage type instead. (support iOS/tvOS/macOS)

    Warning

    This API need polishing. In the future we may choose to create a new View type instead.
